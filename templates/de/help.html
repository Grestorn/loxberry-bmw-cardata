Schnellstart

0. Konto erstellen

  • Klicke auf "+ Konto hinzufügen" in der Kontoleiste
  • Gib einen Namen ein (z.B. "Martins BMW")
  • Klicke auf "Erstellen"
ℹ️ Das Plugin unterstützt mehrere BMW-Konten. Jedes Konto hat eine eigene Konfiguration, Anmeldung und Bridge. Wiederhole die folgenden Schritte für jedes Konto.

1. BMW CarData Client erstellen

  • Öffne MyBMW Fahrzeugübersicht
  • Wähle Fahrzeug → "CarData Client erstellen"
  • Kopiere die Client-ID
  • Aktiviere: "CarData Stream" + "Zugang zur CarData API"
⚠️ Nicht auf "Gerät authentifizieren" drücken!

2. Stream-Benutzername ermitteln

  • Scrolle zu "CarData Streaming"
  • "Datenstream konfigurieren" → "Verbindungsdetails anzeigen"
  • Kopiere den Benutzernamen

3. Daten auswählen

  • "Datenauswahl ändern"
  • Nur benötigte Werte auswählen
  • "Absenden und Stream starten"

4. Plugin konfigurieren

  • Client-ID eintragen
  • Stream Host: customer.streaming-cardata.bmwgroup.com
  • Stream Port: 9000
  • Stream Benutzername eintragen
  • "Konfiguration speichern"

5. Anmeldung durchführen

  • "Registrierung starten"
  • "Zur BMW Anmeldung" → Mit BMW-ID anmelden
  • Falls BMW nach Code fragt: Code aus Plugin kopieren und auf BMW-Webseite eingeben
  • "Tokens abrufen"
ℹ️ Der Anmeldecode bleibt im Plugin sichtbar und kann jederzeit kopiert werden.

MQTT Topics

Format: <prefix>/<VIN>/<attribut>

Standard-Präfix: bmw-<kontoname>

Beispiel:
bmw-martin/WBA1234.../data/vehicle.powertrain.electric.battery.chargeLevel.percent/value

⚠️ BMW sendet nur bei Änderungen Daten. Zum Testen Fahrzeug öffnen.

Loxone Integration

  • LoxBerry MQTT Gateway öffnen
  • Subscription hinzufügen: bmw-martin/# (durch eigenen Konto-Präfix ersetzen)
  • Anleitung: MQTT Schritt für Schritt

Fehlerbehebung

Anmeldung schlägt fehl:

  • Client-ID und Benutzername prüfen
  • Logs auf "Protokolle"-Tab prüfen

Bridge startet nicht:

  • Status muss "Authentifiziert" sein
  • "Token manuell aktualisieren" klicken

Keine Daten in Loxone:

  • Bridge-Status prüfen
  • Daten im BMW Portal ausgewählt?
  • Bridge-Logs prüfen (Anzahl Nachrichten)
  • Fahrzeug öffnen zum Testen

FAQ

Wie lange ist die Anmeldung gültig?
Max. 2 Wochen, automatische Erneuerung alle 10 Min.

Separate Konfiguration pro Fahrzeug?
Nein! Stream enthält automatisch alle Fahrzeuge pro BMW-ID.

Mehrere BMW-IDs?
Ja! Erstelle pro BMW-ID ein eigenes Konto über "Konto hinzufügen".

Was passiert bei LoxBerry-Neustart?
Bridge startet automatisch beim Systemstart.
